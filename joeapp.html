KUNDECASE FOR JOE AND THE JUICE

I dette projekt arbejede vi i gruppe med en virksomheds case hvor vi fik stillet af firmaet Joe and The Juice følgende opgave. "Hvordan kan Joe and The Juice styrke Joe Loyalty-appen?" Projektet tog to uger hvor vi skulle producere en prototype med en forbedret kundeoplevelse.



Følgende værktøjer blev brugt til dette projekt:

    GitHub
    Interview
    HTML og CSS
    FTP og domæne
    Basal test med BERT
    Storytelling og dramaturgi
    Adobe Ai, Ps, Id og Xd.
    Visuelt design, herunder icon design
    Brug af Brackets til at udvikle brugergrænseflader
    Content-tekst, billeder og animationer
